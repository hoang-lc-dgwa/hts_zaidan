【アジア体験コンテストとは…？】

日本人学生を対象に、アジア諸国で働く夢を実現させる支援を目的として、毎年開催しているコンテス卜です。
対象国の3か国で①日本語教育体験　②就業体験　のどちらかを行うインタ一ンシップ（疑似職業体験）の企画書を作成してもらいます。2019年度の対象国は①カンボジア　②マレーシア　③ミャンマーの3か国です。
一次審査(書類選考)で通過者を選考し、二次審査(面接)にて入賞者を決定します。
入賞者10名には賞金20万円を支給し、希望体験を実現してもらいます。


＜選考実施報告＞
一次審査・・・書類選考（当財団指定の企画書）応募締切6月14日（金）
審査委員による書類と体験先交渉手続状況の審査。
6月28日（金）当財団ホームページにて通過者10名を発表。 二次審査・・・7月6日（土）12：30～
