「中国福祉医療人材育成基金」奨学金

当財団では、2008年度より中国の福祉医療人材育成を目的とする奨学金を支給しております。

支給金額及び支給期間（2014年4月から支給）

奨学金種別 中国福祉医療人材育成基金奨学金
支給金額 毎月5万円
支給期間 1年間
募集人数 毎年4月・10月（各5名程度）

応募条件

■応募資格：

1.中国国籍を有する者
2.中国の看護士・福祉士資格を有する者
3.日本語能力試験3級以上
4.中国の大学、大学専科卒以上の新卒者

■応募方法： 当財団指定の中国医療系大学に募集要項と申請書を配布し、現地にて一次選考（書類審査）・二次選考（面接）を実施する
